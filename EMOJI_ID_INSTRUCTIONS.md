# Інструкція: Отримання Emoji ID через Koyeb Logs

## Крок 1: Задеплоїти код на Koyeb
Я вже додав тимчасовий код для логування emoji ID в файл:
`d:\kms\electo_bot\electro_bot\libs\bot\src\lib\notification-bot.service.ts`

Задеплойте оновлений код на Koyeb (як зазвичай).

## Крок 2: Надішліть емоджі боту
Після деплою відкрийте Telegram і надішліть вашому боту **4 емоджі з набору esvitlo_uk** одним повідомленням або окремими повідомленнями:

1. Анімована галочка ✓
2. Анімована лампочка 💡
3. Анімований хрестик ❌
4. Батарейка на зарядці 🔋

## Крок 3: Перегляньте логи Koyeb
Відкрийте логи вашого застосунку на Koyeb. Ви побачите виділений вивід:

```
╔═══════════════════════════════════════════════════════════╗
║ 🎯 CUSTOM EMOJI DETECTED - #1                            ║
║ ID: 5368324170671202286                                   ║
║ From: your_username                                       ║
╚═══════════════════════════════════════════════════════════╝
```

## Крок 4: Скопіюйте ID
Виділіть і скопіюйте **4 рядки з ID**. Наприклад:

```
ID: 5368324170671202286  (галочка)
ID: 5370869711888194012  (лампочка)
ID: 5359785904535774578  (хрестик)
ID: 5370809889929639451  (батарейка)
```

## Крок 5: Надішліть мені
Скопіюйте і надішліть мені ці 4 ID у такому форматі:
```
Галочка: 5368324170671202286
Лампочка: 5370869711888194012
Хрестик: 5359785904535774578
Батарейка: 5370809889929639451
```

## Крок 6: Видалю тимчасовий код
Після отримання ID я:
1. Видалю тимчасовий код логування
2. Інтегрую кастомні анімовані емоджі
3. Додам нагадування про заряджання
4. Конвертую формат з Markdown на HTML

---

**ВАЖЛИВО:** Тимчасовий код логування працює для **будь-яких повідомлень** з кастомними емоджі, тому ви можете їх надсилати в будь-якому порядку - головне, щоб ви знали який емоджі якому ID відповідає.
